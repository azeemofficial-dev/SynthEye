Core Capabilities

Four Eyes.
One Truth.

SynthEye deploys a multi-modal AI pipeline to uncover manipulation across visual media and textual content, simultaneously in real time.

01
IMG
Deepfake Image Detection
Detects AI-generated or face-swapped images using frequency-domain analysis, GAN artifact fingerprinting, and biological consistency checks, even on high-quality fakes.
GAN Fingerprint Face Mesh Frequency FFT Blink Pattern
02
VID
Deepfake Video Analysis
Analyses temporal coherence between frames, lip-sync mismatches, unnatural blinking, and micro-expression inconsistencies across video timelines frame by frame.
Temporal Flow Lip Sync Frame Diff rPPG Signal
03
TXT
Fake News Prediction
Uses transformer-based NLP to classify articles, headlines, and social posts by analysing linguistic patterns, emotional manipulation, and source credibility signals.
Transformer NLP Sentiment Claim Verify Source Graph
04
NET
Misinformation Spread Mapping
Traces information propagation across networks, identifies coordinated inauthentic behaviour, and flags narratives matching known misinformation templates.
Graph Neural Net Bot Detection Narrative Track
Pipeline

From Upload
to Verdict.

Every piece of media passes through a multi-stage verification pipeline designed for speed, depth, and explainability.

01
Ingest
Upload image, video, or article. The system auto-detects content type and routes to the correct module.
->
02
Preprocess
Frame extraction, noise normalisation, metadata stripping, and tokenisation for text inputs.
->
03
Analyse
AI models run in parallel: vision and language components cross-examine the evidence simultaneously.
->
04
Score
Ensemble scoring with confidence intervals, explainability heatmaps, and anomaly localisation.
->
05
Verdict
Clear authenticity score, risk level, flagged regions, and a human-readable explanation report.

About the System

Built to Expose
Synthetic Reality.

In an era where synthetic media erodes public trust, SynthEye is a modular AI system for journalists, researchers, platforms, and security analysts who need to know: is this real?

  • Multi-modal detection for images, video, and text in one unified pipeline.
  • Explainable outputs where each verdict includes evidence, not only a score.
  • Robust baselines designed for practical CPU-first deployment.
  • Modular architecture so detection modules can be swapped as models evolve.
  • Privacy-first processing with no long-term media storage by default.
